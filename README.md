# IDK-interpreter
## Usage
| Short Option | Long Option | Arguments | Debug         |       Description      |
|--------------|-------------|-----------|---------------|:----------------------:|
| `-h`         | --help      |           |               |     Show help page     |
| `-s`         | --stdin     |           |               | Run program from stdin |
| -f           | --file      | `file`    |               |  Run program from file |
| -d           | --debug     |           | `1, 2, 3`     |   Debug file or stdin  |
